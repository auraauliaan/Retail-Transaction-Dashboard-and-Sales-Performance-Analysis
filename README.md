# Retail Transaction Dashboard and Sales Performance Analysis

## Description
This project presents a data analysis and dashboard visualization of retail transaction data to evaluate sales performance, customer behavior, and business trends. The dashboard highlights key metrics such as revenue, profit, transaction volume, and payment methods.

This project was developed as part of an e-learning program by MySkill and was completed on January 21, 2025.

## Dataset
The dataset contains transaction records including product details, customer data, payment methods, and sales metrics.

## Tools
- Python
- Pandas
- Data Visualization Tools (Looker Studio)
